# ruby-mandelbrot

Generates a coloured mandelbrot in the terminal.

Based on an [article](http://eigenjoy.com/2008/02/22/ruby-inject-and-the-mandelbrot-set/) by Nate Murray


## Usage

`bundle install`

`bundle exec ruby main.rb`
